# 快速參考

## 啟動
```bash
cd blog
npm start
```

## 網址
- 首頁：http://localhost:3004
- 大家：http://localhost:3004/for-you
- 我的：http://localhost:3004/my-posts
- 登入：http://localhost:3004/login
- 註冊：http://localhost:3004/register

## 使用者操作

| 操作 | 方式 |
|------|------|
| 發文 | 輸入文字 → 點擊「發布」 |
| 編輯 | 點擊鉛筆圖示 → 修改 → 儲存 |
| 刪除 | 點擊垃圾桶 → 確認 |
| 按讚 | 點擊愛心圖示 |

## 標籤語法
```
#標籤名稱
```
範例：`今天天氣真好 #心情 #好天氣`

## 技術棧
- Node.js + Express
- SQLite (sqlite3)
- bcrypt (密碼加密)
- express-session (會話管理)
