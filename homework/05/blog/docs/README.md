# Threads 風格網誌系統

一個使用 Node.js + Express + SQLite 建構的仿 Threads 風格社交網誌系統。

## 目錄結構

```
blog/
├── app.js           # 主程式
├── database.js      # 資料庫設定
├── package.json     # 專案設定
├── blog.db          # SQLite 資料庫
└── docs/
    └── README.md    # 本文件
```

## 安裝與執行

### 必要環境
- Node.js (建議 v14 以上)
- npm

### 安裝步驟

```bash
cd blog
npm install
```

### 啟動伺服器

```bash
npm start
```

伺服器啟動後，開啟瀏覽器前往：http://localhost:3004

## 功能說明

### 1. 使用者系統 (blog1)

#### 註冊
- 前往 `/register` 頁面
- 輸入用戶名稱和密碼
- 密碼會使用 bcrypt 雜湊加密儲存

#### 登入
- 前往 `/login` 頁面
- 輸入帳號密碼
- 成功後會話會持續 1 小時

#### 登出
- 點擊右上角「登出」按鈕

### 2. 發布貼文

#### 發布新貼文
- 需先登入
- 在首頁的文字框輸入內容
- 點擊「發布」按鈕
- 支援 `#標籤` 語法自動建立標籤

#### 編輯貼文
- 點擊自己貼文旁的編輯圖標（鉛筆圖示）
- 在彈出視窗中修改內容
- 點擊「儲存」完成修改

#### 刪除貼文
- 點擊自己貼文旁的刪除圖標（垃圾桶圖示）
- 確認刪除後貼文會移除

### 3. 貼文顯示

#### 大家 (/for-you)
- 顯示所有使用者的公開貼文
- 按發布時間倒序排列
- 最多顯示 50 篇

#### 我的 (/my-posts)
- 只顯示自己的貼文
- 需登入才能存取

#### 用戶頁面 (/user/:username)
- 點擊任何貼文上的頭像或用戶名稱
- 可查看該用戶的所有貼文

### 4. 標籤系統 (blog2)

#### 建立標籤
- 在發布內容時使用 `#標籤名稱` 語法
- 例如：`今天天氣真好 #心情 #好天氣`

#### 查看標籤
- 點擊貼文中的標籤連結
- 進入 `/tag/:tag` 頁面
- 查看所有包含該標籤的貼文

### 5. 按讚功能

- 點擊愛心圖標進行按讚/取消讚
- 按讚數量會即時更新
- 需登入才能使用

## 資料庫結構

### users 資料表
| 欄位 | 類型 | 說明 |
|------|------|------|
| id | INTEGER | 主鍵，自動遞增 |
| username | TEXT | 用戶名稱，唯一 |
| password | TEXT | 加密後的密碼 |
| created_at | DATETIME | 創建時間 |

### posts 資料表
| 欄位 | 類型 | 說明 |
|------|------|------|
| id | INTEGER | 主鍵，自動遞增 |
| user_id | INTEGER | 發文者 ID，外鍵 |
| content | TEXT | 貼文內容 |
| created_at | DATETIME | 發布時間 |

### likes 資料表
| 欄位 | 類型 | 說明 |
|------|------|------|
| id | INTEGER | 主鍵，自動遞增 |
| user_id | INTEGER | 按讚者 ID |
| post_id | INTEGER | 貼文 ID |
| created_at | DATETIME | 按讚時間 |

### tags 資料表
| 欄位 | 類型 | 說明 |
|------|------|------|
| id | INTEGER | 主鍵，自動遞增 |
| name | TEXT | 標籤名稱，唯一 |

### post_tags 資料表
| 欄位 | 類型 | 說明 |
|------|------|------|
| post_id | INTEGER | 貼文 ID |
| tag_id | INTEGER | 標籤 ID |
| (複合主鍵) | | |

## API 端點

### 認證相關

| 方法 | 路徑 | 說明 |
|------|------|------|
| GET | /login | 登入頁面 |
| POST | /login | 執行登入 |
| GET | /register | 註冊頁面 |
| POST | /register | 執行註冊 |
| GET | /logout | 登出 |

### 頁面路由

| 方法 | 路徑 | 說明 |
|------|------|------|
| GET | / | 重新導向 /for-you |
| GET | /for-you | 大家頁面 |
| GET | /my-posts | 我的頁面 |
| GET | /user/:username | 用戶個人頁面 |
| GET | /tag/:tag | 標籤頁面 |

### API 端點

| 方法 | 路徑 | 說明 |
|------|------|------|
| POST | /api/posts | 發布新貼文 |
| PUT | /api/posts/:id | 編輯貼文 |
| DELETE | /api/posts/:id | 刪除貼文 |
| POST | /api/like/:postId | 按讚/取消讚 |

## 疑難排解

### 無法看到舊貼文
資料庫結構更新後需重啟伺服器，系統會自動建立新的資料表。

### 按讚/編輯沒反應
檢查是否已登入，部分功能需要登入才能使用。

### 標籤沒有顯示
確認內容中使用的是 `#標籤` 語法（無空格），例如：`今天 #心情`
