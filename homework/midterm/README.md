## 專案概述

- **後端伺服器**：Node.js + Express（`server.js`）
- **前端**：`public/index.html`（單一 HTML 檔，Vanilla JS）
- **資料庫**：JSON 檔案儲存（`data/scores.json`）
- **技術棧**：純 HTML + CSS + Vanilla JavaScript，無前端框架
- **唯一外部資源**：Google Fonts（Inter + Noto Sans TC）
- **用戶端儲存**：localStorage（個人進度、分數、錯題記錄）
- **伺服器端儲存**：JSON 檔案（公開排行榜資料）

## 啟動方式

```bash
cd 專案資料夾
npm install    # 安裝依賴（首次）
npm start      # 啟動伺服器
```

開啟瀏覽器前往 **http://localhost:3000** 即可遊玩。

## 公開排名系統

- **後端 API**：Express 伺服器提供 `POST /api/score`（提交）與 `GET /api/rankings`（查詢）端點
- **提交時機**：每次完成關卡後，結果畫面會出現「提交至排行榜」按鈕
- **排名規則**：依正確率（pct）降序 → 正確題數降序 → 時間順序
- **篩選功能**：可按關卡篩選排名，只看特定關卡的成績
- **個人標記**：自己的紀錄在排行榜上會有 👈 標示及底色區別
- **玩家名稱**：可在主選單修改顯示名稱（儲存於 localStorage）
